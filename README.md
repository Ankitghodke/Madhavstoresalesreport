Madhav Store sales report
Description:
This project focuses on analyzing e-commerce sales data using Power BI to provide clear insights into business performance. The dashboard highlights overall sales amount, profit, quantity sold, and average order value, along with detailed analysis across months, states, product categories, sub-categories, customers, and payment modes.
TECHSTACK:
To create this project i have used power bi which is a main data visualization platform used for report creation.
Power query:data transformation and cleaning layers for reshaping and preparing data.
File format:pbix for development and png for dashboard previous.
Data source: 
madhav stores sales(order)&details.csv files.
